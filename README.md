## ✨ Key Features

- **Glassmorphism UI** — frosted-glass nav and cards with `backdrop-filter`
- **Typewriter Effect** — animated role headline in the hero
- **Scroll-Reveal** — content enters on scroll with cubic-bezier easing
- **Image Comparison Slider** — drag to compare before/after course designs
- **ScrollSpy Navigation** — active nav link tracks current section
- **Skill Bars** — animated proficiency indicators
- **Ambient Orbs** — soft blurred gradient depth layers
