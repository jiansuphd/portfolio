## ✨ Enhanced Features (Creative Tech UI)

- **Glassmorphism** – Frosted‑glass effects (`backdrop-blur`) on navigation and project cards
- **Ambient Depth** – Subtle, blurred background mesh gradients for a modern, immersive feel
- **Interactive Animations**
  - Typewriter effect for dynamic role headlines
  - Scroll‑reveal transitions for smooth content entry
  - Image comparison slider for “Before & After” instructional design case studies
- **Responsive ScrollSpy** – Navigation updates the active section as you scroll

---

## 🛡️ Security & Privacy Hardening

- Sensitive configuration folders (e.g., `.obsidian/`) and API keys are excluded via `.gitignore`
- All local development metadata and private tokens are removed to ensure safe public hosting on GitHub
